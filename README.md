# 調査用地図 v1

Googleスプレッドシートにまとめた調査記録を地図上に表示し、スマホで現在地と既存記録を見比べるための、閲覧専用Web地図です。調査ごとにスプレッドシートURLを登録しておくと、上部の調査名プルダウンから切り替えて表示できます。

## 想定ヘッダー

1行目に以下のヘッダーを書いてください。

```text
記録ID, 同定, 記録の種類, 所蔵, 標本ID, 場所, Latitude, Longitude, 年, 月, 日, 表示する日付, 備考
```

`記録の種類` には、以下のいずれかを入力します。

```text
タイプ産地
シノニマイズされた種のタイプ産地
文献記録
本調査
```

`表示する日付` には、ポップアップに表示する日付文字列を入力します。`年`、`月`、`日` は機械的な並べ替え・絞り込み用、`表示する日付` は人が読むための表記、という使い分けを想定しています。

```text
19 Jun. 1985
15–16 Apr. 1987
Jun. 1985
1985
```

## 主な機能

- アプリ画面からGoogleスプレッドシートURLを登録し、端末内に保存できます。
- 登録した調査名をプルダウンで切り替えられます。
- Googleスプレッドシートの内容を読み込みます。
- 背景地図は「航空写真」と「淡色地図」を切り替えられます。
  - 航空写真：Esri World Imagery
  - 淡色地図：地理院淡色地図
- `同定` に基づき、分類群ごとにマーカーの色を自動で変えます。まず分類群数に応じて互いに離れた色だけを選び、そのうえで近い場所に出る別分類群どうしでは同じ色グループを避けます。
- `記録の種類` ごとにマーカーの形を変えます。
  - タイプ産地：星
  - シノニマイズされた種のタイプ産地：四角
  - 文献記録：中が空の丸
  - 本調査：中が塗りつぶされた丸
- 画面上で近いマーカーを代表マーカー1個にまとめます。
- 近接グループに複数の分類群が含まれる場合、代表マーカーを分類群の色で radial（扇形）分割表示します。2分類群なら半円ずつ、3分類群なら1/3ずつ、4分類群以上も等分割します。各扇形は記録の種類に応じ、本調査などの標本記録は塗りつぶし、文献記録は中空表示になります。
- 代表マーカーを押すと、近接記録をポップアップ内の「前へ」「次へ」で切り替えられます。
- 地図の拡大・縮小中はマーカーを一時的に非表示にし、ズーム終了後に近接判定を再計算して再表示します。
- 現在地ボタンを表示します。
- 更新ボタンでデータを再読み込みします。

## ポップアップ表示順

```text
同定
記録の種類
場所
Latitude, Longitude [Google Map]
表示する日付
所蔵
標本ID
記録ID
備考
```

`Google Map` は、その座標を Google Maps で開くリンクです。

## 分類群の色の割り当て

分類群の色は、データ読み込み時に一度だけ決めます。ズームや移動によって色は変わりません。

固定距離は使わず、各記録について「最も近い別同定」を近い順に複数探します。同じ近傍に出やすい分類群どうしでは、同じ色グループをできるだけ避け、色差の大きい組み合わせを優先します。

また、分類群数が色候補数以内の場合は、全体として同じ色を再利用しません。8種程度では、まず Lab 色空間で互いに十分離れた高コントラスト色だけを選んでから割り当てます。

`config.js` の `COLOR_NEAREST_DIFFERENT_TAXA` で、各記録から何種類の近い別分類群を見るかを指定できます。初期値は `5` です。

## 近接マーカーの扱い

`config.js` の `PROXIMITY_PIXELS` で、近接記録とみなす画面上の距離を指定します。初期値は `10` です。

代表マーカーの外形は、以下の優先度で決めます。複数の同定が含まれる場合は、代表マーカー内を分類群の色で radial（扇形）分割表示し、文献記録の扇形は中空、本調査などの標本記録の扇形は塗りつぶしで表示します。

```text
タイプ産地 > シノニマイズされた種のタイプ産地 > 本調査 > 文献記録 > その他
```

## 調査名の登録

この版では、アプリ本体に特定のスプレッドシートを固定登録していません。初回起動時は「調査名未登録」と表示されます。

上部の「追加」から入力フォームを開き、以下をまとめて登録してください。「編集」でも同じフォームで内容を修正できます。入力フォーム内には、このページの用途、必要なヘッダー、公開状態に関する説明も表示されます。

```text
調査名：四国のXystodesmus、関東のParadoxosomatidae など
GoogleスプレッドシートURL：通常の共有URL
シート名：通常は シート1。英語環境で作成した場合は Sheet1 のことがあります。
```

登録情報は、その端末・そのブラウザの `localStorage` に保存されます。別の端末では再登録が必要です。ブラウザのサイトデータを削除すると、登録一覧も削除されます。

スプレッドシートは、このアプリのブラウザから読み込める状態にしておく必要があります。通常のGoogleスプレッドシートURLを登録する場合は、リンクを知っている人が閲覧できる共有設定にしてください。公開CSV URLを登録する場合は、CSVとしてウェブに公開されたURLを使用してください。

登録後は、上部の「調査名」プルダウンで切り替えられます。「編集」は入力フォームで端末内の登録情報を修正します。「削除」は端末内の登録情報だけを削除します。スプレッドシート本体は変更・削除しません。

## 設定

通常は `config.js` の次の項目だけ確認してください。

```javascript
DEFAULT_SHEET_NAME: "シート1",
INITIAL_BASEMAP: "detail",
PROXIMITY_PIXELS: 10,
COLOR_NEAREST_DIFFERENT_TAXA: 5,
```

GoogleスプレッドシートURLは、アプリ画面の「追加」から登録します。公開CSV URLを登録することもできます。

## 公開方法

現在地表示は、スマホブラウザではHTTPS上での利用が基本です。
GitHub Pages、Cloudflare Pages、Netlifyなどに配置して使うことを想定しています。

## detail.zip から反映した点

- 学名の自動斜体化は `detail.zip` の `app-labels.js` の考え方に合わせています。
  - `ord.`、`fam.`、`gen.` を含む場合は通常体
  - `sp.` は通常体
  - `cf.`、`aff.` は通常体
  - それ以外の属名・種小名などを斜体
  - `和名 / 学名` 形式では学名側だけを整形
- 背景地図は、`detail.zip` の航空写真に合わせた Esri World Imagery と、地理院淡色地図を切り替えられるようにしています。
