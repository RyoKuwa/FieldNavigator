# 調査用地図 v1

Googleスプレッドシートの採集記録をスマホで確認するための、閲覧専用Web地図です。

## 想定ヘッダー

以下のヘッダーを前提にしています。

```text
ID, 同定, 記録の種類, 所蔵, 標本ID, 場所, Latitude, Longitude, 年, 月, 日, 日付, 備考
```


## 主な機能

- アプリ画面からGoogleスプレッドシートURLを登録し、端末内に保存できます。
- 登録した調査名をプルダウンで切り替えられます。
- Googleスプレッドシートの内容を読み込みます。
- 背景地図は「航空写真」と「淡色地図」を切り替えられます。
  - 航空写真：Esri World Imagery
  - 淡色地図：地理院淡色地図
- `同定` ごとにマーカーの色を自動で変えます。まず同定数に応じて互いに離れた色だけを選び、そのうえで近い場所に出る別同定どうしでは同じ色グループを避けます。
- `記録の種類` ごとにマーカーの形を変えます。
  - タイプ産地：星
  - シノニマイズされた種のタイプ産地：四角
  - 文献記録：中が空の丸
  - 本研究：中が塗りつぶされた丸
- 画面上で近いマーカーを代表マーカー1個にまとめます。
- 近接グループに複数の同定が含まれる場合、代表マーカーを同定色で radial（扇形）分割表示します。2同定なら半円ずつ、3同定なら1/3ずつ、4同定以上も等分割します。
- 代表マーカーを押すと、近接記録をポップアップ内の「前へ」「次へ」で切り替えられます。
- 地図の拡大・縮小中はマーカーを一時的に非表示にし、ズーム終了後に近接判定を再計算して再表示します。
- 現在地ボタンを表示します。
- 更新ボタンでデータを再読み込みします。

## ポップアップ表示順

```text
同定
記録の種類
場所
Latitude, Longitude [Google Map]
日付
所蔵
標本ID
ID
備考
```

`Google Map` は、その座標を Google Maps で開くリンクです。

## 同定色の割り当て

同定色は、データ読み込み時に一度だけ決めます。ズームや移動によって色は変わりません。

固定距離は使わず、各記録について「最も近い別同定」を近い順に複数探します。同じ近傍に出やすい同定どうしでは、同じ色グループをできるだけ避け、色差の大きい組み合わせを優先します。

また、同定数が色候補数以内の場合は、全体として同じ色を再利用しません。8種程度では、まず Lab 色空間で互いに十分離れた高コントラスト色だけを選んでから割り当てます。

`config.js` の `COLOR_NEAREST_DIFFERENT_TAXA` で、各記録から何種類の近い別同定を見るかを指定できます。初期値は `5` です。

## 近接マーカーの扱い

`config.js` の `PROXIMITY_PIXELS` で、近接記録とみなす画面上の距離を指定します。初期値は `10` です。

代表マーカーの形は、以下の優先度で決めます。複数の同定が含まれる場合は、代表マーカー内を同定色で radial（扇形）分割表示します。

```text
タイプ産地 > シノニマイズされた種のタイプ産地 > 本研究 > 文献記録 > その他
```

## 調査名の登録

この版では、アプリ本体に特定のスプレッドシートを固定登録していません。初回起動時は「調査名未登録」と表示されます。

上部の「追加」から入力フォームを開き、以下をまとめて登録してください。「編集」でも同じフォームで内容を修正できます。入力フォーム内には、必要なヘッダーと公開状態に関する説明も表示されます。

```text
調査名：四国のXystodesmus、関東のParadoxosomatidae など
GoogleスプレッドシートURL：通常の共有URL
シート名：通常は シート1。英語環境で作成した場合は Sheet1 のことがあります。
```

登録情報は、その端末・そのブラウザの `localStorage` に保存されます。別の端末では再登録が必要です。ブラウザのサイトデータを削除すると、登録一覧も削除されます。

スプレッドシートは、このアプリのブラウザから読み込める状態にしておく必要があります。通常のGoogleスプレッドシートURLを登録する場合は、リンクを知っている人が閲覧できる共有設定にしてください。公開CSV URLを登録する場合は、CSVとしてウェブに公開されたURLを使用してください。

登録後は、上部の「調査名」プルダウンで切り替えられます。「編集」は入力フォームで端末内の登録情報を修正します。「削除」は端末内の登録情報だけを削除します。スプレッドシート本体は変更・削除しません。

## 設定

通常は `config.js` の次の項目だけ確認してください。

```javascript
DEFAULT_SHEET_NAME: "シート1",
INITIAL_BASEMAP: "detail",
PROXIMITY_PIXELS: 10,
COLOR_NEAREST_DIFFERENT_TAXA: 5,
```

GoogleスプレッドシートURLは、アプリ画面の「追加」から登録します。公開CSV URLを登録することもできます。

## 公開方法

現在地表示は、スマホブラウザではHTTPS上での利用が基本です。
GitHub Pages、Cloudflare Pages、Netlifyなどに配置して使うことを想定しています。

## detail.zip から反映した点

- 学名の自動斜体化は `detail.zip` の `app-labels.js` の考え方に合わせています。
  - `ord.`、`fam.`、`gen.` を含む場合は通常体
  - `sp.` は通常体
  - `cf.`、`aff.` は通常体
  - それ以外の属名・種小名などを斜体
  - `和名 / 学名` 形式では学名側だけを整形
- 背景地図は、`detail.zip` の航空写真に合わせた Esri World Imagery と、地理院淡色地図を切り替えられるようにしています。
